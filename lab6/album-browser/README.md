# Album Browser — Lab 6: Routing, HTTP & Services 

Angular 17+ SPA demonstrating routing, HttpClient, services, and CRUD operations.

## Setup

```bash
npm install
ng serve
```

Open http://localhost:4200

## Features

- 6 routes with Angular Router
- AlbumService (centralized HTTP layer)
- CRUD: Read, Update (edit title), Delete
- 5 standalone components: Home, About, Albums, AlbumDetail, AlbumPhotos
- TypeScript interfaces (Album, Photo)
- Loading indicators & responsive photo grid
