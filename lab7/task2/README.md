# Lab 7 - Task 2: Object-Oriented Programming in Python

## Project Description
This project demonstrates basic OOP concepts in Python using an animal class hierarchy.

## Classes
- `Animal` - base class
- `Dog` - child class
- `Cat` - child class

## OOP Concepts Used
- Class and object creation
- Inheritance
- Method overriding
- Polymorphism
- Modules and imports

## Files
- `models.py` - contains class definitions
- `main.py` - creates objects and demonstrates program output

## How to Run
```bash
python main.py
